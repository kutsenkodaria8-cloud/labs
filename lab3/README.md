# Лабораторна робота №3

## Опис завдання

- Завантаження датасету з відкритого джерела
- Data Cleaning: видалення дублікатів, заповнення пропущених значень
- Побудова 7 графіків для аналізу залежностей між атрибутами:
  1. Scatter Plot — залежність віку від максимального пульсу
  2. Line Plot — середній холестерин за віком
  3. Гістограма — розподіл пацієнтів по 5 вікових діапазонах
  4. Heatmap — кореляція між атрибутами
  5. Box Plot — холестерин у хворих і здорових
  6. Count Plot — хвороба серця за статтю
  7. Violin Plot — розподіл віку у хворих і здорових

Python 3.10+ Jupyter Notebook
